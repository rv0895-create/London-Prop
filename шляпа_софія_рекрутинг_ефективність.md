---

## 1. ЦІЛЬ

Забезпечити постійний приплив кваліфікованих кандидатів — і цілеспрямований пошук талантів — до офісу, підтримувати систему ефективності команди за стандартами Луцького Пропу, а також виявляти вузькі місця в процесах і пропонувати покращення.

> Стандарти Луцького Пропу — це вихідна точка, а не фінальна. Частина роботи — знаходити де система дає збій або може бути кращою, і доносити це з конкретними пропозиціями.

---

## 2. ЦІННИЙ КІНЦЕВИЙ ПРОДУКТ (ЦКП)

> **Укомплектована команда трейдерів, що системно розвивається: нові кандидати своєчасно проходять відбір і онбординг, а поточні трейдери виконують стандарти дисципліни та демонструють зростання ефективності.**

ЦКП вважається досягнутим, коли:
- у воронці рекрутингу завжди є активні кандидати
- серед кандидатів цілеспрямовано шукаються і відбираються таланти
- нові трейдери виходять на роботу без затримок
- система ефективності (дисципліна, внесок) працює і фіксується
- СЕО має актуальні дані по стану команди
- щомісяця фіксується хоча б одне виявлене вузьке місце в процесах з пропозицією дій

## 3. ФУНКЦІЇ

### 3.1. Рекрутинг

- Розміщення і оновлення вакансій (LinkedIn, Indeed, локальні платформи)
- Первинний скринінг резюме та кандидатів
- Проведення первинних співбесід
- Координація фінальних співбесід із Артемом
- Ведення бази кандидатів (воронка: лід → співбесіда → оффер → вихід)
- Онбординг нових трейдерів (перший тиждень, документи, доступи, знайомство)
- Звіт по рекрутингу для Артема — щотижня

### 3.2. Пошук талантів (пріоритетна функція Лондона)

- Цілеспрямований пошук кандидатів із нестандартним мисленням, аналітичними здібностями або унікальним досвідом — не тільки "стандартний трейдер"
- Окрема воронка для талантів: специфічні канали (університети, математичні спільноти, quant-середовище Лондона)
- Первинна оцінка потенціалу таланту: чи може ця людина генерувати нові підходи
- Передача виявлених талантів Артему для подальшої роботи по R&D-напряму

### 3.3. Ефективність

- Ведення системи дисципліни за стандартами Луцького Пропу
- Відстеження виконання обов'язкових елементів (базове навчання, журнал угод, регламенти)
- Фіксація рівнів внеску трейдерів у розвиток офісу (матеріали, обговорення, тестування)
- Підготовка щотижневого зрізу по ефективності для Артема
- Ескалація системних порушень або ризиків до Артема
- Організація загальних заходів розвитку (якщо впроваджується система Луцького Пропу)

### 3.4. Виявлення вузьких місць

- Регулярний аналіз процесів рекрутингу та ефективності: де є затримки, повтори помилок, незрозумілі зони
- Збір зворотного зв'язку від нових трейдерів по онбордингу — що незрозуміло, що заважає
- Збір зворотного зв'язку від поточної команди по системі дисципліни та розвитку
- Фіксація спостережень у журналі вузьких місць
- Формування конкретних пропозицій для Артема — не просто "є проблема", а "є проблема + варіант рішення"

### 3.5. Адміністрування

- Ведення актуального списку команди та їх статусів
- Підтримка актуальності документів і регламентів офісу
- Комунікація зі штабом Луцького Пропу з питань стандартів (за потреби)

## 4. ІНСТРУМЕНТИ

| Інструмент | Призначення |
|---|---|
| База кандидатів (таблиця / ATS) | Ведення воронки рекрутингу |
| LinkedIn / Indeed / локальні платформи | Пошук кандидатів |
| Чеклист онбордингу | Стандартизований вхід нових трейдерів |
| Таблиця ефективності трейдерів | Дисципліна, внесок, динаміка |
| Регламенти Луцького Пропу | Стандарти, яким відповідає офіс |
| Шаблон тижневого звіту | Для Артема: рекрутинг + ефективність |
| Канал комунікації з Артемом | Щоденна синхронізація та ескалація |
| Окрема воронка талантів | Відстеження нестандартних кандидатів |
| Журнал вузьких місць | Фіксація спостережень і пропозицій щодо покращень |
